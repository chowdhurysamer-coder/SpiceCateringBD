# Spice Catering Ltd. — Website

Modern marketing website for **Spice Catering Ltd.**, Bangladesh's premier
catering company (Dhaka, est. 2006).

## 🌐 Live site

https://chowdhurysamer-coder.github.io/SpiceCateringBD/

## Tech stack

- **Next.js 16** (App Router, static export)
- **Tailwind CSS 4**
- **TypeScript**
- **Embla Carousel** (services gallery)
- **lucide-react** (icons)
- Fonts: Playfair Display (headings) + Montserrat (body)

## Sections

Hero · Stats · About · Services carousel · Tabbed menus · Testimonials ·
Photo strip · Contact form · Footer

## Local development

```bash
npm install
npm run dev      # http://localhost:3000
```

## Build

```bash
npm run build    # static export to ./out
```

## Deployment

Pushes to `main` or the working branch trigger the
[Deploy to GitHub Pages](.github/workflows/deploy.yml) workflow, which
builds the static export and publishes it to GitHub Pages automatically.
